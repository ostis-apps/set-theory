Свойства нечетких множеств
Пусть задано нечеткое множество A c функцией принадлежности μA на универсуме X.

  • α-разрезом нечёткого множества A, обозначаемым как Aα, называется следующее чёткое множество: Aα = {x ∈ X; μA ≥ α}, то есть множество, определяемое функцией принадлежности χ Aα, которая принимает значение 0, если μA < α и 1, если μA ≥ α.
    Для α-разреза истинна импликация α1 < α2 ⇒ Aα1 ⊃ Aα2
  • Нечёткое множество выпуклым тогда и только тогда, когда для каждой пары точек х1,x2 из Х, выполняется условие:
       μA (λх1 + (1 - λ)х2) ≥ min (μA(х1), μA(х2)); где λ ∈ [0,1]
  • Нечёткое множество вогнутым тогда и только тогда, когда для каждой пары точек х1,x2 из Х, выполняется условие:
       μA (λх1 + (1 - λ)х2) ≥ max (μA(х1), μA(х2)); где λ ∈ [0,1]
