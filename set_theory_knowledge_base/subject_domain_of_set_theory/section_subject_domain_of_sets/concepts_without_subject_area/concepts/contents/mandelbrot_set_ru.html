Мно́жество Мандельбро́та — множество точек c на комплексной плоскости, для которых рекуррентное соотношение zn+1=zn2+c задаёт ограниченную последовательность. Иными словами, это множество таких c, для которых существует такое действительное R, что неравенство |zn| < R выполняется при всех натуральных n.
