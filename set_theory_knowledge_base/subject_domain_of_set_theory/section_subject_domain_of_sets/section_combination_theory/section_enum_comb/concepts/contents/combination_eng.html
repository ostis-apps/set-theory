A combination of n things taken k at a time is a set of k elements, selected from an n-element set, in which the order of the elements is not taken into account.

Combinations that differ only in the order of the elements (but not in their composition) are considered the same - this is how combinations differ from arrangements.

The number of all possible k-element subsets of an n-element set is located at the intersection of the k-th diagonal and the n-th row of the Pascal's triangle.
