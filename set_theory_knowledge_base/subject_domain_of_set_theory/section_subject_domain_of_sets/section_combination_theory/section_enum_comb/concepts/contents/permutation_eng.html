In combinatorics, a permutation of a given finite set X = {a1, a2, ..., an} (all elements of X are distinct) is any ordered arrangement of all the elements of X. By grouping these elements in different orders, one can obtain different permutations. In total, from a set of n elements, there can be n! = 1 ⋅ 2 ⋅ 3 ⋅ ... ⋅ n different permutations.
