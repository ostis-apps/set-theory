Verbal formulation:

  1. 1 is a natural number;
  2. The number following a natural number is also a natural number;
  3. 1 does not follow any natural number;
  4. If a natural number a immediately follows both the number b and the number c, then b and c are identical;
  5. (Axiom of induction.) If a certain proposition is proved for 1 (the induction base) and if from the assumption that it is true for a natural number n, it follows that it is true for the next natural number after n (the induction hypothesis), then this proposition is true for all natural numbers.

Mathematical formulation:

The mathematical formulation uses the successor function S(x), which assigns to the number x the next number.

  1. 1 ∈ ℕ;
  2. x ∈ ℕ ⇒ S(x) ∈ ℕ;
  3. ∄ x ∈ ℕ: (S(x) = 1);
  4. (S(b) = a ∧ S(c) = a) ⇒ b = c;
  5. P(1) ∧ ∀n (P(n) ⇒ P(S(n))) ⇒ ∀n ∈ ℕ (P(n)).
