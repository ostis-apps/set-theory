транспозицией конечного множества X = { a1 , a2 , … , an } называется такая его перестановка f, то есть биекция из X в X, что существуют такие индексы i и j, что f ( ai ) = aj, f ( aj ) = ai и f ( ak ) = ak для всех остальных индексов k. Транспозиция называется элементарной, если индексы i и j — соседние.
